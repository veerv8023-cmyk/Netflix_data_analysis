# Power BI Data Analysis Project

This repository contains a Microsoft Power BI project file (**pbi.project 1.pbix**) created for data analysis and visualization. The project presents interactive dashboards and reports to help users understand data trends and performance metrics.

---

##  Project Description

The **pbi.project 1.pbix** file is designed to analyze datasets and convert raw data into meaningful visual insights. It uses Power BI’s powerful tools for data transformation, modeling, and visualization.

This project demonstrates practical business intelligence techniques and supports data-driven decision-making.

---

## 🚀 Features

- ✔️ Interactive dashboards  
- ✔️ Dynamic filters and slicers  
- ✔️ Cleaned and transformed data  
- ✔️ Well-organized data model  
- ✔️ Visual representation of KPIs  
- ✔️ User-friendly layout  

---

## 🛠️ Tools & Technologies

- Microsoft Power BI Desktop  
- Power Query  
- DAX (Data Analysis Expressions)  
- Windows OS
